Project 1: GBDA 101 - Icons

Description

I dedicated this project to Chinese traditional instruments that I have learned since I was younger. The icons are based on the Guzheng, Hulusi, and Pipa. I explored shapes, textures, and colour to give each instrument its own personality while keeping a unified visual style.

Project 2: GBDA 228 – Website Redesign

Description

For this project I redesigned the HVAC Engineering Inc. website to create a more modern, professional, and user-friendly experience. I improved the visual hierarchy, navigation, and overall layout to make information easier to scan and understand. I also introduced a clearer colour palette, new imagery, and interactive elements to better communicate the company’s values and services.

Next Project

Project 3: GBDA 201 – Documentary

Description

In this GBDA 201 project, my group and I created a documentary exploring MBTI (Myers-Briggs Type Indicator) and how it shapes the way people see themselves and relate to others. Through interviews and animation, we examined different perspectives on MBTI, its strengths and limitations, and how people use it in their everyday lives.
